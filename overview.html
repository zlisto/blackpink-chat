Blackpink Chat

A vibe coded AI app

Concept

An interactive web application that provides a unique fan engagement experience. Users can register and hold AI-powered conversations with digital personas of the Blackpink members (and a special guest, Bret "The Hitman" Hart).

Unique Selling Proposition (USP)

  • Personalized Interaction: Moves beyond static content to dynamic, one-on-one conversations.
  • Persistent Memory: Chat history is saved, creating a feeling of an ongoing relationship with the AI persona.
  • User Customization: Power users can modify the AI's core personality, offering a deeper level of engagement.

Core Functionality

  • ✓
    User Authentication: Secure registration and login system using JWT and bcrypt hashing for a personalized and protected user experience.
  • ✓
    AI Chat Personas: Converse with distinct AI models for Jisoo, Jennie, Ros√©, and Lisa, each with a unique system prompt defining their personality.
  • ✓
    Persistent Conversations: Chat history is stored in MongoDB, allowing users to continue conversations where they left off.
  • ✓
    Image Recognition (via GPT-4o): The AI can "see" and respond to images uploaded by the user during a chat.
  • ✓
    Customizable AI Agents: Authenticated users can edit the underlying system prompts to fine-tune the AI's personality and responses.
  • ✓
    Memory Management: Users have full control to delete their chat history with any member, ensuring data privacy.

System Flow

└─ (Handles UI, State, User Input)

└─ (Handles Logic, Auth, API Routing)

[Node.js Backend] <-- CRUD Ops --> [MongoDB] (Users, Chats, Prompts)
[Node.js Backend] -- Sends Prompt --> [OpenAI API]
[OpenAI API] <-- Returns Response -- [Node.js Backend]

Monetization & Growth Strategy

Freemium Model

Offer core chat features for free to attract a large user base. A premium subscription could unlock advanced features.

Potential Premium Features

  • Unlimited conversation history and memory.
  • Access to exclusive or limited-time AI personas.
  • Advanced AI agent customization tools.
  • Future capabilities like AI-generated voice notes.

Strategic Partnerships

Collaborate with entertainment agencies to create official, verified AI chat experiences for other artists, expanding the platform's reach.

Data Analytics

Leverage anonymized conversation data to provide valuable insights into fan sentiment, trending topics, and brand perception for market research.
